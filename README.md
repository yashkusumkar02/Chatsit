# Chatsit
Chatsit is a WhatsApp clone social networking app that supports Android Studio, Firebase &amp; Admob. User can create profiles, create groups, private chat, group chat, send files in chat (Image, video, location, document, audio, GIF, sticker, contacts etc.) User video &amp; audio call one-one and group video and audio call.
